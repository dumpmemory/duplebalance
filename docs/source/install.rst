Install `duplebalance`
*******************************

Prerequisites
=============

The following packages are requirements:

    * ``numpy``
    * ``scipy``
    * ``pandas``
    * ``joblib``
    * ``sklearn``
    * ``matplotlib``
    * ``seaborn``

Installation
============

Clone from GitHub
^^^^^^^^^^^^^^^^^

You can install duplebalance simply by running:

.. code-block:: bash
    
    > git clone https://github.com/ICDE2022Sub/duplebalance.git
    > cd duplebalance
    > pip install .
